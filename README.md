# Wedly

Wedly is an AI-powered wedding planning workspace that helps couples and wedding planners manage the full journey in one place.

The app includes:
- onboarding for couples and planners
- a personal dashboard
- timeline and task planning
- vendor tracking
- budget, shopping, and guest management
- an AI orchestrator and communication agent

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Add your environment variables in `.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
OPENAI_API_KEY=your_openai_api_key
```

3. Start the app:

```bash
npm run dev
```

4. Open it in the browser:

```text
http://localhost:3000
```
